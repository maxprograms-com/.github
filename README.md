# Maxprograms

Maxprograms develops **software infrastructure**, **tools**, and **libraries** built on open standards, with an emphasis on **clean architecture**, **modularity**, and **long‑term maintainability**.

## Areas of work

- Standards‑based infrastructure components and tooling
- Libraries implementing open specifications (XLIFF, TMX, TBX, SRX, GlossML)
- Modular TypeScript and Java solutions for enterprise workflows
- Developer‑friendly, sustainable software architecture

## Principles

- Clean, minimal, future‑proof design
- Standards compliance without unnecessary dependencies
- Transparent versioning and predictable behavior
- Long‑term maintainability over short‑term convenience

## Repositories

This organization hosts Maxprograms' core infrastructure projects, including standards‑based libraries, command‑line tools, and supporting components.
